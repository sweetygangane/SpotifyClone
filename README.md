# SpotifyClone
Spotify Web Player Clone
A simple and responsive Spotify Web Player UI Clone built using HTML and CSS.
This project replicates the basic layout and design of Spotify, including sidebar navigation, cards, and a music player interface.

🚀 Features
🎧 Sidebar navigation (Home, Search, Library)
📂 Playlist & Podcast sections
📊 Cards layout (Recently Played, Trending, Featured)
🎶 Music player UI with controls
📌 Sticky navigation bar
💻 Responsive design (basic)
🛠️ Technologies Used
HTML5
CSS3 (Flexbox & Media Queries)
Font Awesome (Icons)
Google Fonts (Montserrat)
